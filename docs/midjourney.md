# Midjourney 深度指南：从基础到高阶控制

Midjourney 凭借其卓越的艺术表现力和高质量的生成结果，始终在 AI 图像生成领域占据重要地位。截至 2026 年 4 月，Midjourney 已推出 V8.1 Alpha 版本，标志着该平台从"提示词驱动"向"个性化配置驱动"的重大范式转变。本指南系统梳理了从基础提示词构建到高阶参数控制的完整知识体系，涵盖 V6 至 V8.1 的全部核心功能。

## 提示词基础结构与描述词分类

构建一个有效的 Midjourney 提示词通常遵循一套结构化的公式。这个灵活的公式为 AI 提供了明确的指导方向：

`[主题] [动作/场景] [风格/艺术家] [细节/修饰词] [--参数]`

在这一结构中，主题（Subject）是图像的主要焦点，例如"一位中世纪骑士"或"一座未来城市"。动作或场景（Action/Setting）则定义了主题正在做什么或所处的环境。风格与艺术家（Style/Artist）决定了图像的美学影响，例如"宫崎骏电影风格"或"油画"。细节与修饰词（Details/Modifiers）用于补充具体特征、氛围或技术术语，如"戏剧性的电影光效"或"发光的眼睛"。

### 关键描述词库

为了获得更精确的控制，生图师需要掌握不同类别的描述词：

| 类别 | 常用关键词示例 |
| :--- | :--- |
| **光照 (Lighting)** | cinematic lighting, golden hour, soft light, neon glow, dramatic backlighting, volumetric light, rim light, studio lighting |
| **色彩 (Colors)** | monochromatic, pastel palette, vibrant cyberpunk colors, earthy tones, complementary colors, fluorescent, iridescent |
| **镜头 (Camera)** | wide shot, close-up, macro photography, drone shot, anamorphic lens flare, bokeh, tilt-shift, low angle, high angle |
| **材质 (Textures)** | rough wood, smooth metal, silky fabric, intricate lace, glowing crystal, liquid metal, glass, chrome |
| **氛围 (Atmosphere)** | foggy, rainy, snowy, dusty, dreamy atmosphere, eerie mist, epic fantasy, serene |
| **艺术风格 (Art Styles)** | impressionism, surrealism, cubism, art nouveau, baroque, rococo, fantasy art, sci-fi, cyberpunk, pixel art |

## 高级提示词技巧

在掌握基础结构后，进阶用户可以通过特定语法进一步控制生成结果。

**权重调整 (Weighting `::`)**：通过在术语后添加双冒号和数字，可以赋予特定元素更多或更少的相对重要性。数字越高，权重越大。例如，`red car::2 blue car::1` 将使红色汽车在画面中占据更主导的地位。

**负面提示 (`--no`)**：此参数用于明确告诉 Midjourney 在图像中排除哪些元素，这比在正向提示词中使用"不要"或"没有"更为有效。例如，`a medieval town --no buildings, cobblestone`。

**排列组合 (`{ }`)**：使用花括号可以从单个提示词生成多个变体，极大地提高了探索不同概念的效率。例如，输入 `/imagine a {red, blue, green} sports car` 将同时触发三个独立的生成任务。

**文字渲染**：在 V8 中，将需要渲染的文字放入引号中（如 `"HELLO WORLD"`）可以显著提升文字生成的准确性。短语和标签类文字效果最佳。

## 核心命令与参数控制

Midjourney 提供了丰富的命令和参数来调整图像的最终呈现。核心命令如 `/imagine` 用于生成图像，`/blend` 用于混合多张图像的概念，`/shorten` 用于分析冗长提示词并建议精简版本，而 `/tune` 则可生成风格调谐器以创建自定义风格代码。

### 全版本参数速查

在提示词末尾添加的参数（以双连字符 `--` 开头）是精细控制的关键：

| 参数 | 描述 | 示例值范围/用法 | 适用版本 |
| :--- | :--- | :--- | :--- |
| `--ar` | 设置图像的宽高比（默认 1:1） | `--ar 16:9` (横向), `--ar 2:3` (纵向) | 全版本 |
| `--v` | 指定使用的模型版本 | `--v 7`, `--v 8` | 全版本 |
| `--s` | 控制风格化程度（艺术化 vs 写实化） | `0-1000`，V8 推荐高值如 1000 | 全版本 |
| `--c` | 增加初始网格中四张图像的差异性和混沌度 | `0-100` | 全版本 |
| `--q` | 控制细节质量和渲染时间 | `0.25`, `0.5`, `1` (默认), V8 新增 `4` | 全版本 |
| `--seed` | 设定种子值，用于重现一致的视觉噪声 | `0-4294967295` | 全版本 |
| `--iw` | 平衡图像提示与文本提示的权重影响 | `0-3`，V8.1 回归支持 | V6+/V8.1 |
| `--tile` | 生成可无缝拼接的重复图案 | 直接添加在末尾 | 全版本 |
| `--raw` | 减少默认美化效果，呈现更原始的风格 | 直接添加，V8 推荐用于写实 | 全版本 |
| `--weird` | 引入奇特、古怪和意想不到的元素 | `0-3000`，默认为 0 | V6+ |
| `--no` | 负面提示，排除特定元素 | `--no text, watermark` | 全版本 |
| `--sref` | 风格参考图像 | `--sref <URL>` | V6+ |
| `--sw` | 风格参考权重 | `0-1000`，默认 100 | V6+ |
| `--cref` | 角色参考图像 | `--cref <URL>` | V6+ |
| `--cw` | 角色参考权重 | `0-100` | V6+ |
| `--oref` | 全能参考（替代 cref） | `--oref <URL>` | V7+ |
| `--p` | 个性化配置 | `--p [个人代码]` | V7+ |
| `--draft` | 草稿模式，低成本快速探索 | 直接添加 | V7 |
| `--hd` | 原生 2K 高清渲染 | 直接添加，V8.1 默认开启 | V8+ |
| `--sv` | 风格版本，切换风格引擎 | `--sv 7`（默认） | V8+ |
| `--exp` | 实验模式 | 直接添加 | V8+ |

## V8/V8.1 Alpha：范式转变

### V8 的核心架构变化

V8 Alpha（2026 年 3 月发布）是 Midjourney 的一次全面架构重设计，而非简单的版本微调。模型的训练管道、提示词解析方式和默认美学输出都发生了根本性变化。V8 在写实能力上取得了显著突破，皮肤和织物的微纹理更加锐利，景深效果更接近真实镜头，肖像照片的表现尤为出色。人体解剖学的准确性也大幅提升，手部畸变、多余手指等经典 AI 缺陷的出现率大幅降低。

V8 最重要的范式转变在于创作控制从文字提示词转向个性化配置文件。模型在 `--stylize 1000` 配合训练过的个性化配置时表现最佳。这意味着创作者的审美品味——通过长期的图像评分和偏好选择积累——成为了主要的创作输入，而文字提示词则更多地承担内容描述的角色。

### V8.1 的关键改进

V8.1 Alpha（2026 年 4 月发布）在 V8 基础上进行了重要优化。美学风格回归了 V7 的熟悉感，解决了 V8.0 被批评的"过度精致"问题。HD 模式（原生 2K 分辨率）速度提升 3 倍、成本降低 3 倍，并成为默认输出模式。标准分辨率速度提升 50%、成本降低 25%，全质量输出速度已等同于 V7 的草稿模式。新增的"Run as HD"按钮允许将任何标准分辨率作品一键升级为 HD。图像提示（`--iw`）在 V8.1 中重新回归（V8.0 曾暂时移除），Moodboards 和 srefs 变得"超级稳定"。

### V8 推荐工作流

V8.1 的最佳工作流程分为三个阶段：首先在标准分辨率下快速探索大量变体（速度等同 V7 draft），然后在找到满意的构图后点击"Run as HD"升级至原生 2K，最后依赖个性化配置处理美学方向。这种"先量后质"的工作流充分利用了 V8.1 的速度优势。

### Style Creator 与风格控制体系

V8 强化了视觉风格探索工具。Style Tuner（`/tune` 命令）通过生成视觉问卷让用户选择美学偏好，产出唯一的风格代码（如 `--style a1b2c3d4`）。这些代码可以保存、复用和分享，特别适合团队协作中保持视觉一致性。`--sref` 参数从参考图像中提取美学特质（而非内容），配合 `--sw` 权重控制影响强度。在 V8 中，个性化配置（`--p`）成为核心创作方式，模型从用户长期的图像评分中学习审美偏好。

## V7 与 V6 的高级控制特性

在 V6 中引入的 `--cref <URL>`（角色参考）和 `--sref <URL>`（风格参考）彻底改变了工作流。通过配合 `--cw 0-100`（控制角色参考强度，0 仅关注面部，100 包含服饰发型）和 `--sw 0-1000`（风格权重），创作者可以前所未有地保持角色和视觉风格的连贯性。

V7 进一步引入了 `--oref`（Omni Reference，全能参考），它不仅替代了角色参考，还能锚定特定人物、物体或物品，使其在多张图像中保持高度一致。V7 还提供了 `--draft` 草稿模式，以更低的成本和更快的速度进行早期概念探索。

通过熟练运用这些提示词结构、词汇库和高级参数，创作者可以极大地提升对 Midjourney 生成结果的掌控力，创作出令人惊叹的视觉作品。
