# Stable Diffusion 进阶控制：结构与细节的极致掌控

Stable Diffusion（简称 SD）作为开源图像生成模型的代表，以其无与伦比的定制化能力和精准的结构控制闻名于世。从基础的提示词撰写到复杂的 ComfyUI 工作流，再到 ControlNet 和 LoRA 的组合应用，SD 为专业创作者提供了实现像素级控制的可能性。

## 提示词构建与核心参数

与 Midjourney 的自然语言风格不同，Stable Diffusion 更倾向于结构化和基于标签的提示词系统。

### 提示词八大核心类别

构建 SD 提示词时，一个行之有效的流程是参考以下八个类别的检查清单：

1. **主题 (Subject)**：明确你想在图像中看到的核心内容。
2. **媒介 (Medium)**：定义艺术品的材质，如插画、油画、3D渲染或摄影。
3. **风格 (Style)**：指定艺术风格，如印象派、超现实主义或波普艺术。
4. **艺术分享网站**：通过提及 Artstation、Deviant Art 等平台，引导图像向特定审美靠拢。
5. **分辨率与细节**：使用 `highly detailed`、`sharp focus` 等词汇提升图像清晰度。
6. **额外细节 (Sweeteners)**：添加 `sci-fi`、`dystopian` 等氛围词汇修饰图像。
7. **色彩 (Color)**：通过 `monochromatic`、`iridescent gold` 等控制整体色调。
8. **光照 (Lighting)**：使用 `studio lighting`、`cinematic lighting` 等关键照明术语。

### 关键生成参数解析

SD 的生成过程受到多个核心参数的直接影响，理解并合理设置这些参数是获得高质量图像的前提。

**迭代步数 (Sampling Steps)**：决定了模型从初始噪声去噪到最终图像的次数。通常，10-15 步适合快速测试新提示词，25 步是标准生成配置。如果生成包含复杂纹理（如动物毛发或精细面部）的主题，可增加至 40 步。在现代高效采样器下，超过 100 步通常是不必要的，且会浪费计算资源。

**采样器 (Samplers)**：采样器是执行去噪过程的算法。
- **Euler A**：产生边缘较柔和、颜色更平滑的图像，带有“梦幻”感。
- **DDIM 和 DPM Solver++**：速度快，能在 15-25 步内生成边缘清晰、连贯的结果。

**CFG 引导尺度 (CFG Scale)**：平衡模型创造力与遵循提示词的严格程度。
- **2-6**：赋予 AI 更多自由，适合短提示词，但可能偏离预期。
- **7-10**：推荐的默认范围，在创意与引导之间取得良好平衡。
- **10-15**：适用于提示词极其详细且明确的情况。
- **16-20**：通常不推荐，可能导致图像出现不自然的伪影（Artifacts）。

**随机种子 (Seed)**：决定初始随机噪声的数值。相同的种子配合相同的提示词和参数，将生成完全相同的图像。利用这一特性，可以通过固定种子并微调提示词（如改变表情、发色或风格）来精确控制特定元素的改变。

**重绘幅度 (Denoising Strength / Img2img)**：在图生图模式中，决定在原图基础上添加多少噪声。0 代表完全保留原图，1 代表完全被噪声覆盖。0.5-0.75 适合在保留原图构图的基础上创建变体；较低的强度（如 0.25）多次迭代，则能在保持极高相似度的同时改变图像风格。

## 高级提示技巧与语法

SD 支持丰富的提示词语法，允许创作者微调模型对特定词汇的关注度。

**关键词权重**：使用括号可以精确调整词汇的影响力。
- `(keyword:1.5)` 增加 50% 的权重。
- `(keyword:0.5)` 降低 50% 的权重。
- 简写语法：`()` 增加约 10% 权重，`[]` 降低约 10% 权重。

**关键词混合 (Prompt Scheduling)**：语法如 `[keyword1:keyword2:0.5]` 允许在生成过程的特定阶段切换焦点。例如，在总步数的 50% 时，模型停止关注 keyword1，转而关注 keyword2，从而创造出融合的概念。

**负面提示词 (Negative Prompt)**：这是 SD 中极其强大但常被低估的功能。明确告诉模型“不要生成什么”（如 `ugly`, `mutated`, `extra limbs`），往往比在正向提示词中不断堆砌溢美之词更能有效提升图像质量。

## 极致控制：ControlNet、LoRA 与 IP-Adapter

SD 之所以能在专业工作流中占据主导，很大程度上归功于其强大的扩展生态系统。

### ControlNet：结构与姿态的绝对掌控

ControlNet 是一种神经网络架构，它通过引入额外的条件图像，极大地增强了对生成过程的控制。

- **Canny / Line Art / HED**：边缘检测模型，能够精确提取参考图的轮廓线条，并据此生成新图像，完美保留原图结构。
- **OpenPose**：提取人物的骨骼关键点（包括面部表情和手部姿态），使得创作者可以随心所欲地控制生成角色的动作。
- **Depth**：通过深度图理解场景的空间层次，确保生成图像的透视关系与参考图一致。
- **Scribble**：将简单的手绘涂鸦草图转化为高质量的艺术作品。
- **Normal Map / Seg (Segmentation)**：利用法线贴图控制表面凹凸细节，或通过语义分割图控制场景中不同元素的布局。

### LoRA (Low-Rank Adaptation)：风格与概念的微调

LoRA 是一种轻量级的微调模型，通常只有几十兆大小。它可以被视为特定风格、角色、物品或概念的“补丁”。
- 创作者可以同时叠加多个 LoRA，并精确调节各自的权重强度。
- 无论是想要特定的动漫角色、独特的插画风格，还是特定的摄影质感，LoRA 都能提供极具针对性的解决方案。

### IP-Adapter：图像提示的革命

IP-Adapter（Image Prompt Adapter）允许直接使用图像作为提示，而不是仅仅依赖文本。
- 它通过提取参考图像的特征嵌入，引导模型生成在风格、色彩或内容上高度相似的新图像。
- 特别是 **IP-Adapter Face** 模型，专为面部特征设计，能够在不同的场景和风格中保持角色面部的高度一致性，解决了长期以来 AI 生图难以维持同一角色的痛点。

## ComfyUI：专业级节点式工作流

随着 SD 生态的发展，ComfyUI 已成为高级用户的首选界面。它采用基于节点的视觉编程方式，允许创作者将模型加载、提示词编码、采样器、ControlNet、LoRA、以及后期处理（如 Inpainting 局部重绘、Outpainting 图像扩展、Upscaling 高清放大）自由组合。

通过构建复杂的 ComfyUI 工作流，生图师可以实现自动化的批量处理、多阶段的精细渲染，以及极具创意的图像生成管道，真正将 AI 图像生成从“抽卡”转变为可控的工程实践。
